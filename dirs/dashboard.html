About the Dashboard

The App is built to read excel files by using a JavaScript function. We provide both a model and an example of what the dashboard can do, right below. Make sure your excel file follows the format so that Tableau can read and deploy the Dashboard.

Safety

We don't store any files, so your excel should be uploaded each different session for the app.

Question Answer Subject Source Responsible Right/Wrong Quizer
1 When was SPE founded? 1957 SPE SPE 0 Player 1
2 When was USA's first Well drilled? 1859 Non-Technical SPE Timeline 1 Player 2
3 Oil whose API gravity is below 10? Ultra heavy oil Reservoir PetroWiki -1 Player 3

For the "Right/Wrong" column, we consider three values:

  • -1: The player missed the question
  • 0: Nobody buzzed for the question
  • 1: The player got the question right
